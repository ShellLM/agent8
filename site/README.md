# Agent8 Publication Site

Static site for the Agent8 experiment publication.

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

Output in `dist/` directory.

## Structure

```
src/
├── content/posts/    # Blog posts (markdown)
├── layouts/          # Page layouts
└── pages/            # Static pages
public/evidence/      # Evidence files (CSV, MD)
```

## Deploy

Built for GitHub Pages at https://shelllm.github.io/agent8/
